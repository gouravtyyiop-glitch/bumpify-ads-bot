# 🚀 Bumpify Ads Bot

  Powerful Telegram group ad broadcasting bot with multi-account support, real-time analytics, web panel, and AES-256 encrypted session storage.

  ## Quick Deploy
  - **Railway**: Connect this repo → add env vars → deploy
  - **Render**: Use render.yaml  
  - **VPS**: `docker-compose up -d`

  See [SETUP.md](SETUP.md) for full instructions.
